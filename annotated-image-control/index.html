    Try out our new 'annotated image' control

    Aug 14, 2019

    The 4.10.0 release of forms4health delivers a new type of control that allows users to annotate an image - for instance to describe patient injuries on a body map, or to illustrate heart defects.

    Adding an annotated image control to a form in the form designer

    In the designer palette the building block author can choose the image they want for users to add annotations to.

    Using the control

    Introducing a new 'scribble' control

    The 4.9.0 release of forms4health delivers a new type of control that will allow form users to create a free form sketch / diagram of their choice. For instance this could be used to sketch ianatomical parts, adding labels etc. as part of a consultation or it could be used to allow patients to sketch an injury. We call this control scribble.

    New rich text accordion variation and styling improvements

    The 4.12.0 release of forms4health delivers a new variation of the rich text select control, along with styling improvements to the original rich text select control and the master child repeat.

    Rich text select as accordion

    In the designer you can change the rich text control into an accordion by going to the additional settings tab and changing the Display Type to Collapsible
