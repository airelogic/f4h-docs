    Building Block Designer Overview

    Find out more about the Building Block Designer layout and features

    The Building Block Designer is where you can edit the building blocks that make up your form.

    To get to the Building Block Designer go to the forms4health Manage Building Blocks tab.

    • Click Create New to edit a new building block OR
    • Click Import and Edit to edit an existing building block

    Building Block Designer Menu

    On the left hand side, there is a menu of all the data fields (also known as controls) that you can use to build building blocks. You can find out more about these features on the edit building blocks page.

    Editing grids

    Grids can be used within a building block to provide a structured layout for your data fields.

    As shown in the Demographics section of this example building block, grids can be used to create form layouts with two or more data fields being visible side-by-side.

    The grey directional arrows can be used to add new columns and rows alongside existing tiles to control the grid layout. Each row can have multiple columns within it.

    To quickly edit grids, use the Form Structure options at the top of the Building Block Menu.

    Data fields (controls): quick edit

    To quickly edit or move data fields (or controls) use the Cut, Copy and Paste options at the top of the Building Block Menu.

    You can also expand (arrow icon) or delete (bin icon) any data fields that have been edited. The grid (rows and columns) will still be available when a data field is deleted.

    Sections and section controls

    Building blocks can be separated into sections. Sections help to break up the data items and response fields for the form user, and they also allow the form builder to more easily control the settings that apply to data fields (by applying settings to sections).

    Section and Section Title

    • Click the grey box at the top of a section to edit the title
    • Click New Section in the

    Moving your cursor over a section will make the Section Editing Menu visible.

    • The Delete (bin) icon will delete the selected building block section
    • The Section Settings (cog) icon opens the the (see below)
    • The Help (information) icon opens up the Help message editor to provide help information for form users
    • The green arrows can be used to move sections around (up/down or across) and change their order

    In the Section Settings menu there are three tabs:

    • Basic Settings and Formulas
    • Repeated Content
    • Additional Settings
    Basic settings and Formulas

    This tab enables you to rename a section, specify if and when a section is visible to form users (including in diifferent modes e.g. edit, read or PDF) and edit section presentation (e.g. collapsible section).

    Repeated Content

    This tab enables you to specify settings for multiple repeats of content within a section. This saves time editing building blocks for content that requires the same structure or information.

    Additional Settings

    This tab enables section highlighting, which helps to emphasise content in explanatory or information sections. find out more in the highlighting explanation.
