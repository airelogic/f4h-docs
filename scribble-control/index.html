    Introducing a new 'scribble' control

    Jul 17, 2019

    The 4.9.0 release of forms4health delivers a new type of control that will allow form users to create a free form sketch / diagram of their choice. For instance this could be used to sketch ianatomical parts, adding labels etc. as part of a consultation or it could be used to allow patients to sketch an injury. We call this control scribble.
